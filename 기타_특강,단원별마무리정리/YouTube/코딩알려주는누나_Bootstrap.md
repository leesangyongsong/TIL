# 📚 0912_ 부트스트랩(Youtube)

## 📌 코딩알려주는누나
```javascript
// 유튜브 링크
https://youtu.be/3Az_hKsL9L8

// 부트스트랩의 특징
1. 부트스트랩 내의 코드를 그대로 가져와서 간편하게 사용 가능
2. 반응형 웹사이트까지 지원(grid-system)
  - row, col 클래스명으로 그리드 시스템 설정 가능
  - 1줄을 12로 보고 화면 크기별로 설정하여 그리드 시스템 활용
  - ex) col-숫자, col-lg-숫자, col-md-숫자, col-sm-숫자

// 웹사이트 1분만에 만들기
부트스트랩 사이트 navbar 에서 'examples' 클릭
-> 원하는 홈페이지 예시 클릭
-> 마우스 우클릭 후 'View page source' 클릭
-> vs code에 복사, 붙여넣기
-> 부트스트랩 연결 css, javascript 링크 수정!

```
