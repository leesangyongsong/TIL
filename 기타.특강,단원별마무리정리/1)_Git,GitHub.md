## 번외
```
.gitkeep
: 빈 폴더를 만들기 위해 사용
-> git을 효율적으로 관리하기 위해 파일을 기준으로 함
=> 빈 폴더는 의미X, 이름바꾸기 가능

.gitignore
: git에서 추적하지 않는 파일 관리
-> git 모든 하위 디렉토리의 파일을 축적
=> 이름바꾸기 불가능
```
