# 📚 팔로워와 수익화를 부르는 Instagram 운영 전략

## 📌 5장. 팔로워/좋아요를 2배 늘리는 운영 전략
```javascript
5-1. 팔로워, 늘리고 싶다면 이것부터 분석하자

  나를 주로 어떤 사람들이 팔로우하며, 어디서 어떻게 내 게시물을
  발견하고, 내 콘텐츠 중에서도 어떤 콘텐츠를 가장 좋아할까?

  // 인사이트
  : 내 게시물에 대한 팔로워들의 활동(ex. 좋아요, 댓글, 공유, 저장 등)
    을 집게해주는 인스타그램의 기능

  // 인사이트의 특징
  - 팔로워의 전반적인 트렌드를 파악하고 
    타겟에 대한 콘텐츠 성과에 대해 자세히 알 수 있음

  - 비즈니스 또는 크리에이터 계정으로 전환한 이후에
    게시한 콘텐츠에 대한 인사이트만 볼 수 있음

  // 인사이트 확인해야할 3가지
  1) 전체 인사이트 확인하기(비즈니스/크리에이터 계정만 가능)
  2) 스토리 인사이트 확인하기
    - 스토리를 조회한 사람 확인 가능
    - 스토리 반응(통계) 확인 가능
  3) 게시물 인사이트 확인하기
    - 내가 업로드한 게시물(사진&영상) 클릭 후 
      '인사이트 보기' 버튼 클릭
    - 게시물에 대한 인사이트는 2시간 이후부터 집계되므로 
      그 이후에 확인하기
    - 인사이트 구성 
      1. 요약지표 
        - 이 사진을 통해 발생한 좋아요, 댓글, DM, 저장수
      2. 반응지표(더 자극적인 행동)
        - 이 사진을 통해 발생한 
          프로필 방문, 웹사이트, 통화, 이메일 클릭 수
      3. 발견지표(발견한 사람 수와 유입 출처)
        - 도달(Reach)의 경우, 내 게시물을 발견한 계정 수
        - 노출(Impression)의 경우, 내 게시물이 노출된 횟수
      + 이 사진을 통해 발생한 팔로우 수

5-2. 내 계정에 방문할 팔로워는 어디서 찾을까?

  // 인스타그램 가짜 인플루언서들이 하는 일
  1. 팔로워를 구매한다.
  2. 좋아요를 구매한다.
  3. 댓글을 구매한다.

  // HyperAuditor
  : 계정을 분석해 가짜 인플루언서를 알려주는 서비스

  // 유입이 늘어나는 순서
  내가 공략가능한 것부터 실천해나가자!

  프로필 유입 -> 홈 유입 -> 해시태그 유입

  // 해당사항 체크 -> 해당 시 무인도라고 생각하면 됌 ㅋㅋ 냉정하네 
  - 팔로워 1천명 이하
  - 좋아요 평균 300개 
  - 댓글 평균 20개

  // 계정 초기에 '반응률' 확보 필요

  // 프로필을 통한 좋아요/팔로우 획득법
  - profile journey(프로필 여정)
    1. 프로필 방문 
      - 잠재 팔로워에게 좋아요/팔로잉 등 다가가기(기본조건)
    2. 내 게시물 발견
    3. 반응(인지/좋아요)
    4. 프로필 재방문
    5. 팔로잉
      => 프로필을 통한 유입을 늘리기 위해서는 반응해줄만한 계정에
         먼저 찾아가는 좋아요, 팔로잉(선팔)이 중요

  // 전략적으로 내가 먼저 좋아요/팔로잉 해야 하는 계정
  - 잠재 고객 및 잠재 팔로워를 찾아가는 온라인식 방문판매
    1. 나와 관심사가 같은 계정 
      - 경쟁자 계정의 팔로워
      - 경쟁자 계정의 게시물에 좋아요를 한 계정
      - 경쟁자 계정의 게시물에 댓글을 단 계정
    2. 잠재 팔로워 힌트를 주는 계정
      - 잠재 팔로워들이 사용할만한 해시태그를 사용한 계정
      - 잠재 팔로워들이 거주하는 지역을 태그한 계정

  // 선팔/맞팔, 좋아요반사를 해도 팔로워/좋아요가 늘지 않는 이유
    1. 인스타그램을 자주 사용하지 않는 계정을 대상으로 한 경우
    2. 내 콘텐츠에 관심이 없는 사람을 대상으로 한 경우
    3. 타인의 콘텐츠에 반응해본 경험이 없는 계정을 대상으로 하는 경우
    4. 작업 양이 적은 경우

  // 30% - 30% 법칙
    100명에게 팔로밍을 하면 30%(약 30명) 정도가 맞팔을 하고,
    맞팔한 팔로워를 언팔로우하면 30%(약 10명) 정도가 
    내 팔로워로 남는다.

  // 프로그램 작업 업체 선정시 주의점
    1. IP 및 서버 위치(븐다시 국내 IP 및 서버로 작업)
    2. 한 개의 IP당, 한 개의 계정을 작업하는 것이 안전
      - 업체에 작업을 맡기는 시기에는 직접 활동 지양하기
    3. 적정 하루 작업량을 지키기(계정의 생성시기 고려)
    4. 작업량 증가 속도 및 최대 작업량 엄수
    5. 작업시간 제한(1-6시 새벽시간에는 작업 중지)
    6. 각종 스팸 계정 및 키워드/해시태그 필터링
    7. 작업계정(잠재 팔로워)를 선정하는 방식(효율성)
  
  // 계정 반응률 증가를 위한 추천 소통량(1일 기준)
    - 좋아요 : 200개
    - 팔로우 : 100개
    - 언팔로우 : 50개
    - 댓글, DM : 30개(내용 다르게)

  // 사람들이 언팔로우를 하는 이유?
    1. 맞팔을 해주지 않아서
    2. 영업하러 내 계정을 팔로잉한 사람들이 이탈
    3. 내 게시물을 더이상 보고 싶지 않아서
    4. 상대방이 본인의 팔로잉을 정리하고 싶어서
    5. 상대방이 계정을 삭제해서

  // 언팔로우 하는 방법
    1. 팔로우 백하지 않음(맞팔 X)
      - 선팔 작업 후 1~2일 뒤에 모두 언팔
    2. 최근 언팔로우어(팔로우 했다가 언팔로우한 계정)
      - 발견 즉시 바로 언팔
    3. 상호(서로 맞팔하고 있는 계정)
      - 활동이 별로 없다고 판단되는 계정은 가차없이 언팔
        내 게시물에 좋아요/댓글 등의 활동이 없는 계정은 정리!
    4. 팬(상대방만 나를 팔로우한 계정)
      - 가끔씩 각 계정에 들어가서 좋아요, 댓글 등의 팬 관리 활동 필요

5-3. 절대 상단 노출되지 않는 게시물의 특징

  // 홈을 통한 좋아요/팔로우 획득법
  - Home Journey(홈 여정)
    1. 인스타그램 접속
    2. 내 게시물 발견
      - 내 게시물에 참여한 경험이 있어야만 함
    3. 반응(인지/좋아요)

  // 나의 반응률(engagement rate)에 가장 영향을 주는 사람들.
  - 평소에 꾸준히 좋아요/댓글로 관리하기
    => 일반계정의 경우, 평균적으로 팔로워의 5%가 반응
       알고리즘에 의해 좋은 계정으로 평가받기 위해서
       10% 이상의 팔로워에게 반응을 얻는 것이 좋음

    Tip. 새 게시물을 올린 후에 이전 게시물에 
         반응했던 계정들 대상으로 좋아요/댓글/팔로잉 해주기

  // 2019 인스타그램이 발표한 홈 피드 알고리즘 구성
    - 사진도달범위를 팔로워의 7%까지로 제한
    - 피드는 각각의 사용자에게 맞춤화되어 있음
    - 피드에 제일 먼저 표시되는 것은 가장 많이 참여하는 게시물 계정
    - 기타요소: 게시물의 적시성, 사용 빈도, 팔로잉 계정 수 등
    - 최근에 피드 순위를 변경하지 않았으며, 내가 팔로잉하는 사람의
      소식을 숨기지 않음, 계속 스크롤하면 계속 볼 수 있음
    
    => 내 팔로워 모두가 홈 탭에서 내 사진을 볼 수 있는 것은 아니다!
       스토리를 통해 공감받으면 초기 반응을 확보할 가능성이 커진다.

  // 반응률
    반응률 = (좋아요 + 댓글 + 공유 + 저장 등)/내 팔로워 수 * 100%

    반응률은 인스타그램 알고리즘이 '좋은 계정/콘텐츠'라고 판단하는
    기준이 되며, 팔로워가 적은 초기에는 비교적 반응률을 높이기 쉽기
    때문에 홈 유입 관리를 부지런히 해놓도록 한다.

5-4. 노출 최적화를 위한 해시태그 작성 노하우

  // 해시태그 6가지 필수구성
    1. 제품 또는 서비스를 알리는 해시태그
      ex) 스웨터, 카페, 브랜드명
    2. 본인/고객을 정의하는 해시태그
      ex) 직장인, 커플, 대학생, 육아맘
    3. 특별한 이벤트 또는 시즌을 나타내는 해시태그
      ex) 벛꽃축제, 불꽃놀이
    4. 위치를 사용하는 해시태그
      ex) 강남맛집, 성수역카페, 청담동미용실
    5. 일상 관련 해시태그
      ex) 월요일, 월요팅, 헬요일, 불금, 불토
    6. 관심사 해시태그
      ex) 책, 영화, 테니스, 골프, 인테리어
    
  // 2019년 팔로워가 가장 많은 해시태그
    [공부] - 공스타그램, 스터디플래너, 노트필기, 공스타
    [취미] - 슬라임, 고양이, 여행에미치다, 방꾸미기
    [케이팝] - 방탄소년단, 아이유, 아이린, 키드밀리
    [패션&스타일] - 데일리룩, 아메카지룩, 스트릿패션, 미니멀룩
    [일상] - 글귀스타그램, 글귀, 영화명대사, 위로글귀, 감성글귀, 공감글귀, 힐링 글귀

  // 해시태그 사용법의 모든 것
    네이버 광고 https://searchad.naver.com/
    : 네이버 검색엔진에서의 월간 검색량 확인

    해시태그 LAB https://tag.mediance.co.kr/
    : 해시태그별 반응도, 트렌드 지수 파악 가능

    해시태그 추천 https://displaypurposes.com/
    : 해시태그 추천, 로컬 해시태그, 유사해시태그 등 확인 가능

  // 수강생들 해시태그
    그로우커넥트, 그로커, 그로커챌린지  

```
