# 📚 3단위기간 E-learning

## 📌 과제
```
1) 원격 저장소(https://github.com/rhello/hello.git)를 
  로컬에 클론(clone)
  => $ git clone https://github.com/rhello/hello.git

2) 새로운 브랜치(feature)를 checkout 명령으로 생성
  => $ git checkout -b feature

3) 브랜치(feature)에 파일(hello.txt)을 수정하여 
   커밋 메시지("[feature] edit:hello.txt")로 반영
  => $ git commit -m "[feature] edit:hello.txt"

4) 3) 실행 후 원격 저장소에 저장
  => $ git push origin feature


<참고문서>
https://mylko72.gitbooks.io/git/content/branch/checkout.html

```
